# Features Guide

## Core Features

### Activity Discovery
- Browse 2,900+ activities from North Vancouver Recreation Commission
- Filter by category, age range, cost, location, and dates
- Search activities by name, description, or instructor
- View detailed activity information including schedules and requirements

### Categories
- **Aquatics**: Swimming lessons, water safety, competitive programs
- **Arts**: Visual arts, music, drama, crafts
- **Sports**: Team sports, individual athletics, martial arts
- **Dance**: Ballet, hip-hop, contemporary, cultural dance
- **Camps**: Day camps, overnight camps, specialty programs
- **Early Years**: Parent-child programs, preschool activities
- **Youth**: Teen programs, leadership, career exploration

### Child Profiles
- Create profiles for each child with birthdate and interests
- Age-appropriate activity recommendations
- Track registered activities per child
- Activity history and preferences

### Family Features
- **Favorites**: Save activities for later
- **Activity Sharing**: Share activity schedules with co-parents
- **Calendar View**: See all scheduled activities
- **Reminders**: Get notified about registrations and activities

### Calendar Enhancements
- Week and month views
- Shared children calendar overlay
- Quick add activities with FAB button
- Long-press to reschedule activities
- Native iOS Calendar sync
- Activity reminders and push notifications
- Conflict detection for overlapping activities
- Bulk operations (select, delete)
- Print/share calendar as image

## User Interface

### Modern Dashboard
- Personalized activity recommendations
- "New This Week" section
- Quick access to favorites
- Child-specific activity suggestions

### Activity Cards
- Visual thumbnails with category icons
- Price and availability at a glance
- Day(s) of week badges (highlighted in pink)
- Time and location information
- Registration status (open/waitlist/closed)

### Navigation
- Bottom tab navigation (Explore, Favourites, Friends & Family, Profile)
- Top bar with Activities, Filters, and Calendar tabs
- Deep linking support for activity details

## Registration Flow

1. **Browse**: Find activities through search or browsing
2. **Select**: View activity details and availability
3. **Register**: Link to NVRC registration (external)
4. **Track**: Add to child's activity list
